# 🖱️ Virtual Mouse Pro

A Python-based Virtual Mouse using OpenCV, MediaPipe, PyAutoGUI and Pycaw.

## Features

- Hand Tracking
- Mouse Cursor Control
- Left Click Gesture
- Volume Control
- Zoom In
- Zoom Out

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pycaw

## Installation

```bash
pip install opencv-python mediapipe pyautogui pycaw comtypes
```

## Run

```bash
python hand_tracker.py
```

## Controls

- Index Finger → Move Mouse
- Thumb + Index → Left Click
- Thumb + Middle → Zoom In
- Thumb + Ring → Zoom Out
- Thumb ↔ Index Distance → Volume Control
